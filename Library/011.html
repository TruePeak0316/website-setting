各類所得扣繳常見問題與實務重點一次說清楚

撰文： 彭裕峰 會計師


在公司營運過程中，只要有「給付所得」的行為，就很可能會涉及「扣繳」。不少企業主與承辦人常有這樣的疑問：

  • 為什麼付錢給對方，還要先幫他繳稅？
  • 哪些費用一定要扣繳？哪些可以不用？
  • 扣繳之後還要申報嗎？
  • 扣錯、漏扣，會不會被罰？
以下整理實務上最常遇到的各類所得扣繳問題，用白話方式一次說清楚。

一、什麼是「各類所得扣繳」？

簡單說，扣繳就是由給付方代替收款方，先預扣一部分稅款，並繳給國稅局。
也就是說：

  • 公司是「扣繳義務人」
  • 收款的人是「所得人」
  • 國稅局是「最終收稅者」

這個制度的目的，在於確保稅收能即時、確實被掌握，而不是等到個人年底申報時才發現漏稅。


二、哪些情況最常涉及扣繳？

以下是企業最常遇到的扣繳所得：

  1. 薪資所得
    • 正職員工薪水
    • 兼職、工讀生薪資
    是否要扣繳，取決於：
    • 是否達到免扣繳標準
    • 是否為本國人或外國人
    • 是否已填寫免稅聲明文件

  2. 執行業務所得（顧問費、律師費、會計師費）
    3. 只要是：
    • 律師、會計師、設計師
    • 顧問費、審查費、稿費
    舉例：
    貴公司支付律師一筆費用來擬書狀，支付他報酬 30,000 元。依規定這類報酬要扣繳 10% 所得稅，你在付款時就要先扣掉 3,000 元，
    只支付 27,000 元給律師，接著把那 3,000 元繳交國稅局。並在給付隔年1月底之前申報該所得總額給國稅局。

  3. 租金支出
    4. 公司支付房租給自然人房東時，通常需要：
    • 扣繳 10% 所得稅
    • 另計補充保費
    這是實務上最常被忽略、也最容易被補稅的項目之一。

  4. 獎金、紅利、其他給付
    5. 例如：
    • 年終獎金
    • 專案獎金
    • 非固定性給付

三、常見扣繳錯誤有哪些？

在實務查核中，最常見的錯誤包括：

  1. 該扣沒扣：以為對方年底自己會申報
  2. 扣錯稅率：本國人、外國人混用
  3. 只扣稅、不申報：有繳錢但沒申報扣繳憑單
  4. 逾期申報或繳納：產生滯報、滯納金與罰鍰

四、扣繳一定很麻煩嗎？

其實不然。
只要在一開始就做好：

  • 所得性質判斷
  • 身分別確認
  • 稅率適用
後續作業反而單純，也能有效降低被補稅、裁罰的風險。

五、二代健保（補充保費）常見問題

很多企業在處理扣繳時，最容易忽略的，其實不是所得稅，而是二代健保補充保費。

  1. 什麼是二代健保補充保費？
    2. 二代健保補充保費，是在「特定所得」發生且達起扣點時，由給付方代為扣取一定比例的健保費，繳交給健保署。
    目的在於補充一代健保沒有包含的所得類型。

  2. 哪些情況常會產生補充保費？
    3. 實務上，企業最常遇到的包括：
    • 租金支出（支付給自然人房東）
    • 兼職人員、非固定薪資
    • 獎金、紅利、一次性給付
    • 執行業務所得（顧問費、設計費等）
    • 利息所得
    • 股利所得
    是否需要繳納，關鍵在於：
    • 是否屬於健保署規定的補充保費所得項目
    • 是否達到起扣門檻

  3. 常見誤解與風險
    4. 實務上最常見的誤解包括：
    • 「已經扣繳所得稅，就不用補充保費」（錯）
    • 「金額不大，應該不用管」（錯）
    • 「對方自己有健保，公司不用處理」（錯）
    補充保費屬於給付當下就應處理的法定義務，若未扣取，日後仍可能被追繳，且無法向對方補收。

  4. 為什麼二代健保特別容易出問題？
    5. 原因通常不是刻意不繳，而是：
    • 與所得稅扣繳規則不同
    • 稅與費分屬不同機關（國稅局 vs 健保署）
    • 實務上沒有在第一次給付時被提醒
    因此，二代健保往往是在查帳或補資料時才一次被發現。

六、誠峰怎麼看

我們在實務上常見的問題，並不是客戶刻意違規，而是：

  • 不清楚哪些費用算所得
  • 不知道公司要負扣繳或申報責任
  • 沒有人在一開始提醒風險

扣繳不是單純的行政作業，而是稅務風險管理的一環。在公司成長的過程中，越早建立正確觀念，越能避免日後一次補繳多年的壓力。若有其他扣繳相關的問題請聯繫誠峰會計師事務所。
